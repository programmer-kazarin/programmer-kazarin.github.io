Варианты обучения

Java Backend с нуля

  • В данном курсе мы изучим:
    • Алгоритмы и структуры данных
    • Синтаксис языка Java + стандартные библиотеки (Java Core)
    • Самый популлярный java фреймворк для создания веб-приложений (Spring Framework)
    • Базы данных (PostgreSql)
    • Шаблоны проектирования
    • и многое другое (напр. GIT, maven, docker)
  • Cделаем pet-проект, а может не один
  • В конце курса проведем тестовое интервью и оформим резюме
  • Формат обучения
    • Онлайн уроки 1 или 2 раза в неделю. Я буду рассказывать теорию, давать практические задания, отвечать на вопросы
    • Домашние задания
    • Вне занятий общение в телеграме
  • Результат - вы готовы к работе джуниор разработчиком
  • Цена: 3 тыс. за урок (1й бесплатно)

Помощь в трудоустройстве на позицию мидл разработчика

  • Для кого:
    • Вы изучили Java Core и Spring, но не знаете, что с этим дальше делать
    • Возможно, Вы уже начали искать работу на джун и мидл позиции
  • В рамках данной услуги мы будем готовиться к собеседованиям:
    • Проведем мок-собес и определим слабые стороны
    • Разберем такие темы, знание которых обычно требуется от мидл разработчика:
      • CI/CD
      • Микросервисная архитектура
      • Workflow: scrum, agile
      • Gitflow
    • Составим резюме, если его нет. Если есть подкорректируем.
    • Проработаем легенду и научимся рассказывать о предыдущем месте работы
  • Наша цель - устроиться на работу и пройти испытательный срок
  • Цена: 5 тыс. одна консультация + 30% от офера
