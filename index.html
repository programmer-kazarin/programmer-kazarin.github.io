Варианты обучения

Начальный общий

  • Курс находится в разработке, доступно в тестовом режиме

Для кого

  • Хотите стать программистом, но не знаете с чего начать

Что будем делать

  • В данном курсе мы изучим общие темы для любых специальностей, такие как:
    • языковые конструкции, например, переменные, функции, циклы и т.д. (C, java, js)
    • работа с файлами и сетью (bash, HTTP)
    • система контроля версий GIT
    • ознакомимся с основами верстки (html, css)
    • ознокомимся с основами баз данных (sql, JDBC)
  • Cделаем pet-проект, а может не один

Результат

  • изучите основы программирования, алгоритмы и структуры данных
  • сможете определиться с тем какая специальность вам ближе - фронтенд, бекенд или что-то другое

Начальный java backend

  • Курс находится в разработке, доступно в тестовом режиме

Для кого

  • Вы уже определились, что хотите стать java-backend разработчиком
  • Возможно Вы уже знакомы с какими-то языками программирования

Что будем делать

  • В данном курсе мы изучим:
    • Java Core
    • Spring Framework
    • PostgreSql
    • Kafka
    • CI/CD
    • Шаблоны проектирования
  • Cделаем pet-проект, а может не один
  • В конце курса проведем тестовое интервью и оформим резюме

Результат

  • вы готовы к работе джуниор разработчиком

Джун в шкуре мидла - поиск работы

Для кого

  • Вы изучили Java Core и Spring, но не знаете, что с этим дальше делать
  • Возможно, Вы уже начали искать работу на джун и мидл позиции

Что будем делать

  • В рамках данной услуги мы будем готовиться к собеседованиям:
    • Проведем мок-собес
    • Ревью/составление резюме
    • Проработка легенды
  • Сопровождение на испытательном сроке

Результат

  • Наша цель - устроиться на работу и пройти испытательный срок

Формат обучения

  • Созвон 1 или 2 раза в неделю
  • Домашние задания

Мой опыт

  • В разработке с 2015 года на различных проектах:
    • ННК (Роснефть): фулстек разработчик (джун, мидл)
    • ИТ-транс проект: java backend (мидл)
    • Magenta Technology: java backend (мидл)
    • ResolutionPoint: java backend (мидл)
    • Иннотех: java backend (сеньор)
